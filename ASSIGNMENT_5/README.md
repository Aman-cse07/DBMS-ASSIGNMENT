# 📘 DB Lab – 5
Data Retrieval from Multiple Tables Using JOINs

This project focuses on performing SQL JOIN operations to retrieve meaningful information from multiple related tables in a college database system.

## 🎯 Objective

To understand and apply INNER JOIN queries using primary and foreign key relationships without modifying the database structure.

## 🗂️ Tables Used
Student
Faculty
Course
Enrollment
Department
## 🛠️ Tasks Performed
Retrieved data from multiple tables using INNER JOIN
Used table aliases for better readability
Displayed:
Student & Course details
Faculty & Course relationships
Enrollment information
Department-wise records
Applied filtering, sorting, and analytical queries
Used aggregate functions like COUNT() and MAX()

INNER JOIN

Multi-table Relationships
Filtering & Sorting
Aggregate Functions
Relational Database Queries
