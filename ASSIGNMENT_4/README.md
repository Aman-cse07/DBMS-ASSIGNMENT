# 📘 DB Lab – 4
Data Retrieval Using SQL (Read-Only Queries)

This project is based on performing read-only SQL queries on a college database. The lab focuses on retrieving, filtering, sorting, and analyzing data without modifying the original database tables.

### 🎯 Objective
To understand and practice SQL data retrieval operations using:
SELECT
WHERE
ORDER BY
LIMIT
Column Aliases (AS)
Derived/Computed Output

### 🗂️ Tables Used
Student
Faculty
Course
Enrollment
🛠️ Tasks Performed
Displayed records using column aliases
Retrieved data using conditions (WHERE)
Sorted records in ascending and descending order
Limited output using LIMIT
Generated computed values such as:
Student Age from DOB
Updated Credits
Email Domain Extraction

Data Retrieval in SQL
