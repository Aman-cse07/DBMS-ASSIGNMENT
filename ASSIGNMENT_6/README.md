# 📘 DB Lab – 6
Data Aggregation using GROUP BY and HAVING

This project focuses on performing SQL aggregate operations to summarize and analyze data stored in a college database system.

## 🎯 Objective

To understand and apply aggregate functions, GROUP BY, and HAVING clauses for generating meaningful reports without modifying the database.

## 🗂️ Tables Used
Student

Faculty

Course

Enrollment

## 🛠️ Tasks Performed

Used aggregate functions like:

COUNT()

MAX()

MIN()

AVG()

SUM()

Applied GROUP BY for grouped analysis

Used HAVING clause for filtering grouped results

Performed aggregation with INNER JOIN queries

Generated analytical reports such as:

Department-wise student count

Semester-wise enrollments

Course-wise enrolled students

Faculty course count

## 📚 Concepts Covered

Aggregate Functions

GROUP BY Clause

HAVING Clause

Data Analysis using SQL

JOIN with Aggregation
